# Vue專案包備忘
vue專案備忘紀錄

## npm 佈建

### 全域套件
```
npm i -g gulp-cli
npm i -g nodemon
```

### 開發套件
```
npm i --save-dev babel-core babel-loader babel-preset-es2015 css-loader del ejs file-loader gulp gulp-concat gulp-ejs gulp-flatten gulp-less gulp-livereload gulp-nodemon gulp-uglify less less-loader raw-loader should style-loader url-loader vue webpack webpack-stream babel-preset-stage-0
```
